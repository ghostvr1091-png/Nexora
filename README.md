# 👑 Ghost VR's Ultimate Discord Bot

## Deploy to Railway (Free, 24/7)

1. Go to railway.app → New Project → Deploy from GitHub
2. Upload these files to a GitHub repo first
3. Set environment variable: DISCORD_BOT_TOKEN = your token
4. Done! Bot runs 24/7 forever.
